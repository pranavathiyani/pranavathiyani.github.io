Question 1

You are provided with a file proteins.fasta containing multiple protein sequences. Write a Python program with separate functions to perform the following tasks:

(a)

Create a function parse_fasta(filename) that:

• Parses the FASTA file
• Returns a dictionary with structure: {protein_id: sequence}
• Prints all protein IDs
(b)

Create a function analyze_proteins(protein_dict) that calculates for each sequence:

• Length of the sequence
• Find and print the ID of the longest protein
• Percentage of hydrophilic residues (R, K, H, D, E, S, T, N, Q, Y, C)
• Rank all proteins by hydrophilic percentage (highest to lowest) and print results in a formatted table
(c)

Create a function find_motifs(protein_dict, pattern) that:

• Uses regular expressions to find all occurrences of the motif pattern GXXSX (where X = any amino acid)
• For each match, print: Protein ID, Position (start index), and the matched sequence
Example output format:
Protein: P12345 | Position: 23 | Motif: GASSG
