Understanding the Streamlit Flow

How Streamlit Works:

import streamlit as st

st.write("This runs EVERY time")

x = st.slider("Choose a number:", 0, 10)

st.write(f"You chose: {x}")
st.write("This also runs EVERY time")

# The script runs from top to bottom on every interaction!

Important Concepts:

  • Script reruns on every widget interaction
  • No need for callbacks or event handlers
  • State persists between reruns (we'll see this later)
  • Fast and efficient rerun mechanism

FASTA Analyzer - Part 2 (Analysis)

if sequences:
    st.success(f"Loaded {len(sequences)} sequence(s)")
    
    # Analyze each sequence
    results = []
    for record in sequences:
        seq = str(record.seq)
        
        # Calculate metrics
        length = len(seq)
        gc_count = seq.count('G') + seq.count('C')
        gc_percent = (gc_count / length * 100) if length > 0 else 0
        
        a_count = seq.count('A')

        g_count = seq.count('G')
        c_count = seq.count('C')

            'ID': record.id,
            'Length': length,
            'GC%': round(gc_percent, 2),
            'A': a_count,
            'T': t_count,
            'G': g_count,
            'C': c_count
        })
    
    # Display results table
    df = pd.DataFrame(results)
    st.subheader("Analysis Results")

# Cache data loading
@st.cache_data
def parse_fasta(fasta_string):
    """Parse FASTA and return list of records"""
    sequences = list(SeqIO.parse(StringIO(fasta_string), "fasta"))
    return sequences

@st.cache_data
def analyze_sequences(sequences):
    """Analyze sequences - only computed once per unique input"""
    results = []
    for record in sequences:
        seq = str(record.seq)
        results.append({
            'id': record.id,
            'length': len(seq),
            'gc': (seq.count('G') + seq.count('C')) / len(seq) * 100
        })
    return results

Common Pitfalls & Solutions

Common Mistakes:

1. Not handling empty inputs:

# BAD
df = pd.read_csv(uploaded_file)  # Crashes if None

# GOOD
if uploaded_file is not None:
    df = pd.read_csv(uploaded_file)
else:
    st.warning("Please upload a file")

2. Forgetting sequence validation:

# BAD
gc = seq.count('G') + seq.count('C')  # Fails on invalid chars

# GOOD
valid_chars = set('ATGC')
if set(seq.upper()) <= valid_chars:
    gc = seq.count('G') + seq.count('C')
else:
    st.error("Invalid DNA sequence")

3. Not using session state for state persistence:

# BAD - Resets every time
counter = 0

# GOOD
if 'counter' not in st.session_state:
    st.session_state.counter = 0

4. Creating plots without figure objects:

# BAD
plt.plot(data)  # Can cause issues

# GOOD
fig, ax = plt.subplots()
ax.plot(data)
st.pyplot(fig)

import streamlit as st
from Bio import SeqIO
from io import StringIO

def safe_parse_fasta(fasta_string):
    """Safely parse FASTA with error handling"""
    try:
        sequences = list(SeqIO.parse(StringIO(fasta_string), "fasta"))
        
        if not sequences:
